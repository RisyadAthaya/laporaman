# LaporAman
Deskripsi singkat

## How to Run
1. Clone this repository from terminal using the following command
   ```bash
   $ git clone https://github.com/RisyadAthaya/laporaman.git
   ```
2. Install all the required dependencies using the following command in the project root directory
   ```bash
   $ npm install
   ```
3. Run the website using the following command
   ```bash
   $ npm run dev
   ```
